Unidad 1: Arquitecturas de Cómputo

Introducción: La arquitectura de computadoras es el conjunto de principios, diseños y estructuras que permiten el funcionamiento de un sistema informático. Se encarga de definir cómo trabajan e interactúan los componentes internos de una computadora, incluyendo el procesador, la memoria, los buses y los dispositivos de entrada y salida. Además, establece la forma en que se ejecutan las instrucciones, se procesan los datos y se administran los recursos del sistema.

1.1 Modelos de Arquitecturas de Cómputo

Clásicas

Definición: La arquitectura clásica o arquitectura Von Neumann fue propuesta por John Von Neumann en la década de 1940 y es considerada la base de las computadoras modernas. Este modelo utiliza una memoria compartida para almacenar tanto las instrucciones de los programas como los datos que serán procesados. La CPU accede a toda esta información mediante un sistema de buses, ejecutando las instrucciones de manera secuencial. Su diseño permitió simplificar el funcionamiento de las computadoras y facilitar el desarrollo de sistemas informáticos.

En pocas palabras: Los datos y programas se guardan en la misma memoria para que la CPU pueda procesarlos.

Segmentadas

Definición: La arquitectura segmentada, también conocida como Pipeline, divide el procesamiento de instrucciones en diferentes etapas independientes. Cada etapa realiza una parte específica del proceso, permitiendo que varias instrucciones sean ejecutadas simultáneamente en distintas fases. Este método mejora el rendimiento del procesador, ya que aprovecha mejor los recursos internos y reduce el tiempo necesario para ejecutar programas.

De forma simple: El procesador divide el trabajo en partes para ejecutar varias tareas al mismo tiempo.

Multiprocesamiento

Definición: El multiprocesamiento consiste en utilizar dos o más procesadores dentro de un mismo sistema informático para ejecutar múltiples tareas simultáneamente. Este modelo se emplea cuando un solo procesador no es suficiente para manejar grandes cargas de trabajo o procesos complejos. Gracias al multiprocesamiento, las computadoras pueden mejorar su velocidad, eficiencia y capacidad de procesamiento.

Dicho de otra manera: La computadora usa varios procesadores para trabajar más rápido y realizar más tareas al mismo tiempo.

1.2 Análisis de los Componentes

Definición: Los componentes de una computadora son todos los elementos físicos y electrónicos que permiten el funcionamiento del sistema informático. Estos componentes trabajan en conjunto para procesar información, ejecutar programas, almacenar datos y permitir la interacción con el usuario. Algunos componentes se encuentran dentro del gabinete y otros funcionan como dispositivos externos conectados al sistema.

En pocas palabras: Son todas las partes que permiten que una computadora funcione.

1.2.1 Arquitecturas

Arquitectura RISC

Definición: La arquitectura RISC (Reduced Instruction Set Computer) es un modelo de diseño de procesadores que utiliza un conjunto reducido de instrucciones simples y optimizadas. Este tipo de arquitectura busca que las instrucciones puedan ejecutarse rápidamente, generalmente en un solo ciclo de reloj. Gracias a esto, los procesadores RISC suelen ofrecer un mejor rendimiento y eficiencia energética en tareas repetitivas y sistemas modernos.

Más sencillo: Usa instrucciones simples y rápidas para mejorar el rendimiento del procesador.

Arquitectura CISC

Definición: La arquitectura CISC (Complex Instruction Set Computer) utiliza un conjunto amplio de instrucciones complejas capaces de realizar varias operaciones en una sola instrucción. Este modelo facilita la programación y reduce la cantidad de instrucciones necesarias para ejecutar ciertas tareas. Sin embargo, el procesador requiere una estructura más compleja para interpretar y ejecutar dichas instrucciones.

En términos simples: Usa instrucciones más complejas para realizar más tareas en menos pasos.

1.2.2 Unidad Central de Procesamiento (CPU)

Definición: La Unidad Central de Procesamiento o CPU es el componente principal encargado de ejecutar instrucciones y procesar datos dentro de una computadora. Se considera el cerebro del sistema porque coordina todas las operaciones realizadas por los diferentes componentes. Además de ejecutar programas, la CPU administra cálculos matemáticos, operaciones lógicas y el flujo de información dentro del equipo.

En pocas palabras: Es el componente principal que procesa y controla toda la información de la computadora.

Unidad Aritmética Lógica (ALU)

Definición: La Unidad Aritmética Lógica, conocida como ALU, es una sección interna de la CPU especializada en realizar operaciones matemáticas y lógicas. Entre sus funciones se encuentran las sumas, restas, multiplicaciones, comparaciones y operaciones booleanas como AND y OR. Estas operaciones son esenciales para ejecutar instrucciones y resolver procesos dentro del sistema informático.

Más sencillo: Es la parte de la CPU que realiza cálculos y operaciones lógicas.

Unidad de Control (UC)

Definición: La Unidad de Control es el componente de la CPU encargado de coordinar y supervisar todas las operaciones realizadas por la computadora. Su función principal es interpretar las instrucciones almacenadas en memoria y enviar señales de control a los demás componentes para asegurar que cada operación se ejecute correctamente y en el orden adecuado.

Dicho de otra manera: Es la parte del procesador que organiza y controla el funcionamiento de toda la computadora.

Registros

Definición: Los registros son pequeñas memorias internas ubicadas dentro de la CPU que almacenan temporalmente datos, instrucciones y resultados de operaciones. Debido a su gran velocidad de acceso, permiten que el procesador trabaje de manera más eficiente al reducir el tiempo necesario para obtener información importante durante la ejecución de programas.

En términos simples: Son memorias muy rápidas dentro del procesador donde se guarda información temporal.

Buses

Definición: Los buses son canales de comunicación encargados de transportar datos, instrucciones y señales eléctricas entre los distintos componentes de la computadora. Gracias a ellos, la CPU puede comunicarse con la memoria, dispositivos de almacenamiento y periféricos del sistema.

En pocas palabras: Son las vías por donde viaja la información dentro de la computadora.

1.3 Memoria Principal

Definición: La memoria principal es el espacio de almacenamiento temporal utilizado por la computadora para guardar los datos y programas que están siendo utilizados en ese momento. Esta memoria permite que la CPU acceda rápidamente a la información necesaria para ejecutar procesos y tareas del sistema.

Más sencillo: Es la memoria que usa la computadora mientras está trabajando.

Memoria RAM

Definición: La memoria RAM (Random Access Memory) es una memoria principal de acceso rápido utilizada por la computadora para almacenar temporalmente los datos y programas que están siendo utilizados en ese momento. Su función principal es permitir que el procesador acceda rápidamente a la información necesaria para ejecutar aplicaciones y procesos del sistema operativo. La RAM trabaja de manera constante mientras la computadora está encendida y su contenido se elimina automáticamente al apagar el equipo, por lo que se considera una memoria volátil.

La capacidad y velocidad de la memoria RAM influyen directamente en el rendimiento general de la computadora, especialmente en tareas multitarea, videojuegos, edición multimedia y aplicaciones exigentes.

En pocas palabras: Es la memoria temporal donde la computadora guarda la información que está usando en ese momento para trabajar más rápido.

Memoria ROM

Definición: La memoria ROM (Read Only Memory) es un tipo de memoria permanente utilizada para almacenar instrucciones esenciales necesarias para el arranque y funcionamiento básico de la computadora. A diferencia de la RAM, la ROM no pierde su contenido cuando el equipo se apaga, por lo que se considera una memoria no volátil.

En esta memoria se almacenan programas importantes como el BIOS o firmware del sistema, los cuales permiten que la computadora inicie correctamente y reconozca los componentes de hardware instalados.

En pocas palabras: Es una memoria permanente que guarda instrucciones importantes para iniciar la computadora.

Memoria Caché

Definición: La memoria caché es un tipo de memoria de alta velocidad ubicada entre la memoria RAM y el procesador. Su función principal es almacenar temporalmente los datos e instrucciones más utilizados para que la CPU pueda acceder a ellos rápidamente. Esto ayuda a reducir tiempos de espera y mejora significativamente el rendimiento general del sistema.

De forma simple: Es una memoria rápida que ayuda al procesador a encontrar información más rápido.

1.4 Módulos Entrada-Salida (E/S)

Definición: Los módulos de Entrada y Salida son componentes encargados de controlar la comunicación entre la computadora y los dispositivos externos, como teclados, impresoras, discos duros y monitores. Estos módulos ayudan a gestionar las diferencias de velocidad y formato entre la CPU y los periféricos.

En pocas palabras: Permiten que la computadora se comunique con dispositivos externos.

E/S Programada

Definición: La Entrada y Salida programada es un método en el que la CPU controla directamente las operaciones de transferencia de datos entre la memoria y los dispositivos periféricos. Durante este proceso, el procesador permanece supervisando constantemente las operaciones de comunicación.

Más sencillo: La CPU controla manualmente el intercambio de información con los dispositivos.

E/S Mediante Interrupciones

Definición: La Entrada y Salida mediante interrupciones permite que los dispositivos periféricos avisen a la CPU cuando terminan una operación o necesitan atención. Esto evita que el procesador tenga que esperar constantemente y mejora el rendimiento general del sistema.

Dicho de otra manera: El dispositivo le avisa al procesador cuando necesita atención o termina una tarea.

1.5 Acceso Directo a Memoria (DMA)

Definición: El Acceso Directo a Memoria, conocido como DMA (Direct Memory Access), es un mecanismo utilizado en la arquitectura de computadoras que permite transferir datos directamente entre los dispositivos de entrada/salida y la memoria RAM sin que la CPU tenga que intervenir constantemente en el proceso. Gracias a este método, el procesador puede continuar ejecutando otras tareas mientras la transferencia de información se realiza de manera automática, mejorando considerablemente el rendimiento y eficiencia del sistema.

El DMA es muy utilizado en dispositivos que manejan grandes cantidades de información, como discos duros, tarjetas de red, audio y video, ya que reduce la carga de trabajo del procesador y acelera la transferencia de datos.

En pocas palabras: Permite mover datos entre dispositivos y memoria sin usar constantemente el procesador.

1.6 Buses

Definición: Los buses del sistema son estructuras de comunicación utilizadas para transferir datos, direcciones y señales de control entre los diferentes componentes internos de la computadora. Son esenciales para el funcionamiento del sistema, ya que permiten la interacción entre la CPU, la memoria y los dispositivos periféricos.

En pocas palabras: Son canales que conectan todos los componentes de la computadora.

Tipos de Buses

Definición: Existen distintos tipos de buses dependiendo de la información que transportan y su función dentro del sistema. Los principales son el bus de datos, encargado de transferir información; el bus de direcciones, utilizado para indicar ubicaciones de memoria; y el bus de control, que coordina las operaciones del sistema.

Más sencillo: Cada tipo de bus tiene una función específica dentro de la computadora.

Estructura

Definición: La estructura de buses define la forma en que están organizadas las conexiones y rutas de comunicación dentro del sistema informático. Esta organización influye directamente en la velocidad y eficiencia de transferencia de datos entre los componentes.

De forma simple: Es la manera en que los buses están organizados y conectados.

Jerarquía

Definición: La jerarquía de buses organiza los diferentes buses del sistema según su velocidad, prioridad y función. Algunos buses trabajan directamente con la CPU y otros se encargan de conectar dispositivos secundarios o periféricos.

En pocas palabras: Organiza los buses según su importancia y velocidad.

1.7 Interrupciones

Definición: Las interrupciones son señales que provocan que la CPU detenga temporalmente la ejecución normal de un programa para atender un evento importante generado por hardware o software. Este mecanismo permite responder rápidamente a situaciones como la entrada de datos, errores del sistema o solicitudes de dispositivos periféricos.

En términos simples: Son avisos que hacen que la CPU atienda algo urgente antes de continuar trabajando.

Técnica de Poleo (Polling)

Definición: La técnica de poleo o polling es un método utilizado por la CPU para revisar constantemente el estado de los dispositivos de entrada y salida con el objetivo de detectar si alguno necesita atención o si una operación ha finalizado. En este método, el procesador consulta repetidamente cada dispositivo mediante ciclos de revisión continuos.

Aunque es una técnica sencilla de implementar, puede consumir recursos importantes del procesador, ya que la CPU dedica tiempo a supervisar dispositivos incluso cuando estos no requieren atención inmediata.

En pocas palabras: La CPU revisa continuamente los dispositivos para saber si necesitan realizar alguna operación.

Sincronización E/S (Entrada y Salida)

Definición: La sincronización de Entrada y Salida (E/S) es el mecanismo encargado de coordinar la comunicación entre la CPU y los dispositivos periféricos para garantizar que el intercambio de información se realice correctamente. Debido a que los dispositivos de entrada y salida trabajan a velocidades distintas al procesador, es necesario utilizar métodos de sincronización que permitan evitar errores o pérdida de datos.

La sincronización E/S permite que la transferencia de información se realice de manera ordenada y eficiente dentro del sistema computacional.

Más sencillo: Coordina la comunicación entre la CPU y los dispositivos de entrada y salida para evitar errores y mejorar el funcionamiento del sistema.

Conclusión de la Unidad 1

La arquitectura de computadoras es una parte fundamental de los sistemas informáticos, ya que permite comprender cómo se organizan y comunican los componentes internos de una computadora para ejecutar instrucciones y procesar información. A través del estudio de los modelos de arquitectura, la CPU, las memorias, los buses, los módulos de entrada y salida, el acceso directo a memoria y las interrupciones, es posible entender el funcionamiento general de los sistemas de cómputo modernos.

Cada uno de estos elementos cumple una función importante dentro del sistema, trabajando de manera coordinada para garantizar un procesamiento eficiente, rápido y estable. Además, el desarrollo de nuevas tecnologías en arquitectura de computadoras ha permitido mejorar significativamente el rendimiento y capacidad de los equipos actuales, haciendo posible ejecutar aplicaciones cada vez más complejas en distintos ámbitos tecnológicos, científicos y empresariales.
